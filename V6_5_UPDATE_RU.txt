DNX STORE V6.5 — ТОЧЕЧНОЕ ИСПРАВЛЕНИЕ ФИЛЬТРА И КЕЙСА
=====================================================

ИСПРАВЛЕНО
-----------
1. Фильтр каталога теперь физически скрывает неподходящие карточки через display: none !important.
2. Скрытые карточки исключаются из доступности и кликов, их NFT-анимации ставятся на паузу.
3. Все подходящие карточки остаются в сетке без пустых мест, а их NFT-анимации запускаются снова.
4. В рулетке кейса квадратная серая область изображения теперь действительно обрезается по скруглённым углам.
5. Исправлена причина проблемы: квадратное изображение больше не размещается внутри высокого прямоугольного img-элемента.
6. Убрана лишняя рамка и прямоугольный фон вокруг цены на карточке кейса. Остались число и Telegram-звезда.

КАК УСТАНОВИТЬ
--------------
1. Распакуйте dnx-store-stars-final-v6.5.zip.
2. Откройте репозиторий https://github.com/denixl-11/dnx-store.
3. Нажмите Add file -> Upload files.
4. Загрузите все файлы ИЗ папки V6.5 с заменой старых файлов.
5. Нажмите Commit changes.
6. В Render дождитесь автоматического Deploy или нажмите Manual Deploy -> Deploy latest commit.
7. Новые Environment-переменные и ручная миграция базы не нужны.

ПРОВЕРКА
--------
1. Полностью закройте мини-приложение Telegram и откройте заново.
2. Магазин -> Каталог -> выберите Chill Flame: все товары с другим name должны исчезнуть.
3. Проверьте, что оставшиеся Chill Flame продолжают двигаться.
4. Сбросьте фильтр: все карточки должны вернуться, их анимации должны возобновиться.
5. Откройте Кейсы: цена 249 и звезда должны быть без отдельной капсулы/квадрата.
6. Откройте кейс: внутренний серый квадрат каждой картинки должен иметь видимые плавные скругления во всех четырёх углах.

Остальная логика, цены, шансы кейсов, платежи и игровая механика не изменялись.
