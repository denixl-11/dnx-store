# DNX Store

Telegram Mini App магазина NFT-подарков: игра, каталог, кейсы, инвентарь и история.

## Валюта и пополнение

- Внутренний баланс, цены, ставки и выигрыши — целые Telegram Stars.
- Прямая оплата — официальный Telegram invoice с валютой `XTR`.
- Альтернативное пополнение — TON Connect по фиксированному курсу `1 TON = 85 Stars`.
- Рубли отображаются как вариант `Скоро` и не имеют платёжной логики.

## Интерфейс

- Адаптивная safe area для Android, iOS, Telegram Desktop и ноутбуков.
- Ровно две карточки в строке в каталоге, кейсах и инвентаре.
- Фильтр каталога по полю `name` NFT (например, Chill Flame или Big Year) с возможностью сброса.
- Квадратное игровое поле, выровненное по ширине с кнопками.
- Эталонная сине-фиолетово-розовая Telegram-звезда из предоставленного WEBP встроена непосредственно в интерфейс и одинаково выглядит на iOS, Android и компьютере.
- Денежные значения во всех разделах оформлены в порядке `число → звезда`.
- Нижняя навигация примыкает к нижней границе экрана, а системная safe area находится внутри панели.
- В карточке предмета ссылка Telegram находится только в отдельной кнопке сверху слева.
- Пульсирующие космические линии и тонкие молнии во время ожидания игроков.
- 20 контрастных трёхцветных игровых палитр без зелёного, плавная интерполяция градиентов и разделительные контуры полигонов.
- Плавная камера `1.78×`, следующая за мячом все последние три секунды и возвращающая общий план до подсветки победителя.
- Карточки каталога лениво загружают официальный анимированный TGS из `nft_link`; при любой ошибке остаётся исходная статичная картинка.
- Ring, Diamond и Cup в инвентаре не открывают характеристики; остальные NFT показывают `model` только внутри характеристик.
- Премиальные уведомления о пополнении внутри Mini App и в чате бота.

## Основные файлы

- `index.html` — интерфейс Mini App.
- `main.py` — Telegram-бот, API, Neon и обработчики платежей.
- `cosmic-bg.png` — общий фон.
- `game-waiting-bg.png` — оформление игрового поля ожидания.
- `history-empty-bg.png` — оформление пустой истории.
- `app-icon.png`, `tonconnect-manifest.json` — TON Connect.
- `tgsticker.js`, `tgsticker-worker.js`, `rlottie-wasm.js`, `rlottie-wasm.wasm`, `pako-inflate.min.js` — локальный официальный Telegram TGS-рендерер.
- `TELEGRAM_TGS_NOTICE.md` — происхождение файлов TGS-рендерера.
- `DEPLOYMENT_RU.md` — подробная инструкция.

## Render

- Build Command: `pip install -r requirements.txt`
- Start Command: `python main.py`
- Health Check Path: `/health`
- Instances: `1`

Seed-фразы и приватные ключи проекту не нужны.
