DNX STORE V8.9 OPT.7 — УСТАНОВКА

1. Распакуйте архив dnx-store-stars-final-v8.9-opt.7-nft-case-fix.zip.
2. Откройте GitHub-репозиторий проекта и нажмите Add file → Upload files.
3. Перетащите ВСЕ файлы из распакованной папки с заменой существующих.
4. Нажмите Commit changes.
5. Откройте Render → ваш Web Service → Manual Deploy → Deploy latest commit.
6. Дождитесь статуса Live.
7. Полностью закройте мини-приложение Telegram и откройте заново. Если осталась старая версия, очистите кэш Telegram или добавьте к URL мини-приложения параметр ?v=8.9-opt.7.

ЧТО ПРОВЕРИТЬ

1. Игра работает как в предыдущей исправной версии — её механика не изменялась.
2. Нажмите NFT в каталоге и в профиле: окно открывается сразу, есть лёгкий отклик, анимация проигрывается один раз.
3. У NFT, купленного в каталоге, в профиле нет кнопки «Купить».
4. Откройте кейс: во время ожидания и рулетки у анимированных предметов фон тёмно-серый #242424, без Telegram-узора.
5. При загрузке анимации сначала показывается нулевой кадр модели на #242424, затем полноценная анимация.

ВАЖНО

- Не загружайте архив целиком как один файл: на GitHub должны быть заменены файлы проекта внутри архива.
- Environment-переменные Render менять не требуется.
- После деплоя сервер и клиент должны отдавать api_release 8.9-opt.7.
