DNX STORE V8.9-OPT.12 — УСТАНОВКА

Что исправлено
1. NFT снова запускает одну полную анимацию при каждом появлении на экране.
2. После окончания цикла остаётся первый видимый кадр модели, а не пустой фон.
3. Уход с экрана и возврат создаёт свежий проигрыватель и повторяет один цикл.
4. В модальном окне первый цикл начинается при открытии; тап или наведение повторяет цикл.
5. В содержимом кейса тап или наведение также повторяет один цикл.
6. Все анимированные награды кейса имеют единый фон #242424 и масштаб модели 70/30.
7. Рулетка использует зафиксированный кадр модели в том же масштабе 70/30.
8. Убран конфликт скрытого прогрева кейсов с видимыми NFT-анимациями.
9. Ограничен объём кэша кадров, чтобы длительная работа не вызывала лаги.

Важно
- Игра и её расчёты в этой версии не изменялись.
- База данных, переменные Environment и миграции не меняются.
- Загружать нужно все файлы из ZIP целиком, а не только index.html.

Как установить через GitHub
1. Распакуйте ZIP на компьютере.
2. Откройте github.com и войдите в свой аккаунт.
3. Откройте репозиторий denixl-11/dnx-store.
4. Нажмите Add file -> Upload files.
5. Перетащите ВСЕ файлы и папки из распакованной папки. Старые файлы с одинаковыми именами должны замениться.
6. В поле Commit changes напишите: V8.9 opt.12 animation and case fix.
7. Нажмите Commit changes.

Как развернуть на Render
1. Откройте dashboard.render.com.
2. Откройте Web Service вашего магазина.
3. Перейдите в раздел Deploys.
4. Если автоматический деплой не запустился, нажмите Manual Deploy -> Deploy latest commit.
5. Дождитесь статуса Live.
6. Откройте Settings и проверьте, что Build Command и Start Command остались прежними.
7. Environment не изменяйте.

Как исключить старый кэш Telegram
1. После статуса Live полностью закройте мини-приложение.
2. Закройте Telegram из списка запущенных приложений.
3. Запустите Telegram снова и откройте магазин.
4. Версия содержит новый идентификатор 8.9-opt.12, поэтому tgsticker.js и worker будут загружены заново.

Обязательная проверка
1. Каталог: дождитесь одного цикла у всех видимых NFT; прокрутите их за экран и верните — цикл должен повториться.
2. Перейдите в Кейсы и вернитесь в Каталог — видимые NFT должны снова проиграть один цикл.
3. Откройте NFT каталога — должен проиграться один цикл. Нажмите на изображение — цикл повторяется.
4. Откройте NFT инвентаря и корзины и выполните ту же проверку.
5. Откройте кейс: у всех анимированных наград одинаковый тёмно-серый фон, без узора и дополнительного квадрата.
6. Прокрутите содержимое кейса вниз и обратно — появившиеся карточки проигрывают один цикл.
7. Нажмите на карточку награды — один цикл повторяется.
8. Запустите кейс: в рулетке модели не увеличены, вокруг модели остаётся около 30% свободного пространства.

Если после деплоя виден старый интерфейс
1. Проверьте в Render, что развернут именно последний commit.
2. Проверьте логи запуска: API release должен быть 8.9-opt.12.
3. Повторно закройте Telegram полностью, а не только мини-приложение.
4. Не смешивайте файлы этой версии с файлами V8.9-opt.10/11.
