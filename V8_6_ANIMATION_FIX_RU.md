# DNX Store V8.6 — исправление NFT-анимаций

Версия исправляет остановку TGS на первом кадре и унифицирует поведение во всех разделах.

## Поведение на компьютере

- Простое появление карточки на экране не запускает анимацию.
- При новом наведении мыши NFT проигрывает ровно один полный цикл и замирает.
- Пока указатель остаётся над карточкой, повторный цикл не начинается.
- После ухода указателя и нового наведения проигрывается ещё один цикл.

## Поведение на телефоне и планшете

- Когда NFT впервые появляется в видимой области, проигрывается один цикл.
- Пока карточка остаётся на экране, цикл не повторяется.
- После прокрутки карточки за пределы экрана и возвращения она снова проигрывает один цикл.
- После смены вкладки, применения фильтра или возвращения в каталог/инвентарь карточка снова проигрывает один цикл.

Это поведение применяется к каталогу, корзине, инвентарю, содержимому кейсов и окну NFT.

Размер модели во всех анимированных квадратах уменьшен примерно до 65%, чтобы фон и узор оставались хорошо видны. В кейсах узор по-прежнему скрыт, а фон равен `#242424`.

## Установка

1. Откройте репозиторий GitHub.
2. Нажмите **Add file → Upload files**.
3. Загрузите все файлы из папки V8.6 с заменой старых.
4. Создайте commit, например `Fix NFT animation V8.6`.
5. В Render нажмите **Manual Deploy → Deploy latest commit**.
6. Дождитесь успешного запуска в **Logs**.

Новые переменные Environment и новые миграции базы для этого исправления не требуются.
